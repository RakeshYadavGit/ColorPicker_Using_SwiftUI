# ColorPicker Using SwiftUI

A customizable color picker component built using SwiftUI. This project demonstrates how to use native SwiftUI views to create an interactive color picker for iOS/macOS apps, supporting live previews and easy integration.

## Features

- Interactive color selection
- Live color preview
- Easy integration into any SwiftUI project
- Example usage and customization

## Getting Started

### Prerequisites

- Xcode 12 or later
- Swift 5.3 or later
- iOS 14/macOS 11 or later (for SwiftUI support)
